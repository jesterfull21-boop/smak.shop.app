# Smak Shop 💨

Telegram Web App магазин для продажи электронных сигарет

## 🚀 Функции
- Каталог товаров
- Корзина
- Бронь товаров
- Отправка заказов в Telegram

## 🛠 Технологии
- React
- Node.js
- Telegram Bot API

## 📦 Запуск проекта

Frontend:
cd frontend
npm install
npm start

Backend:
cd backend
npm install
node server.js
